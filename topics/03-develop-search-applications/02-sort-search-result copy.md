- Check out the [deployment](../../deployment/single-node/README.md) section to spin up an Elasticsearch cluster.

# Example
Search for data in the `flights` index and sort the result by the `ORIGIN` field in descending order.

# Reference
- https://www.elastic.co/guide/en/elasticsearch/reference/current/search-request-sort.html

# Code
#### Sort search result
```json
GET /flights/_search?size=10
{
  "query": {
    "match": {
      "UNIQUE_CARRIER_NAME": "Airline"
    }
  },
  "sort": [
    {
      "ORIGIN": {
        "order": "desc"
      }
    }
  ]
}
```
